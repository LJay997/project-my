# Markdown 按键指南

## 标题
- `# 空格` + 文字 —— 一级标题
- `## 空格` + 文字 —— 二级标题
- `### 空格` + 文字 —— 三级标题

## 强调
- `**加粗文字**` 或 `__加粗文字__` —— 加粗
- `*斜体文字*` 或 `_斜体文字_` —— 斜体
- `~~删除线~~` —— 删除线

## 列表
- 无序列表：`- 空格` 或 `* 空格` + 内容
- 有序列表：`1. 空格` + 内容

## 链接和图片
- 链接格式：`[链接文字](链接地址)`
- 图片格式：`![替代文字](图片地址)`

## 代码
- 行内代码：使用反引号 `` `代码` `` 包裹
- 多行代码块：
语言
多行代码内容
## 引用
- `> 空格` + 引用文字

## 分割线
- 输入 `---` 或 `***` 或 `___` 单独一行

---

**小贴士**
- 快速加粗文字：选中文字后按 `Ctrl + B`（部分编辑器支持）
- 快速插入链接：选中文字后按 `Ctrl + K`（部分编辑器支持）
